# True Star Executive Dashboard

Executive Dashboard for property management KPIs and metrics.

## Features

- Real-time KPI tracking from webhook integrations
- Property filtering and aggregation
- Modern UI with shadcn components
- Production-ready webhook integration

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

This project is configured for Vercel deployment. The build command is set to `npm run vercel-build`.
